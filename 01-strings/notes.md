# String

- e.g `let forname = "Knud"`
- e.g `let lastname = "Larsen"`
- e.g `let fullname = forname + " " + lastname `

####Template literals

Template literals are literals delimited with backtick (`) and can contain variabels

- e.g `` `${forname} Ole `${lastname}` ``

#### String methodes

- 'Hello'.indexOf('e')
- 'Hello'.chatAt(1)
- 'Hello'.lastIndexOf('l');
- 'Hello'.includes('e');
- 'Hello'.length;
